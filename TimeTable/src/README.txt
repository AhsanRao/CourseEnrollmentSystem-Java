To compile and run the program from the command line, follow these steps:

Open a terminal or command prompt.

Navigate to the directory containing the Java files and CSV file.

Compile the Java files using the following command:
	
	javac Main.java ManagementSystem.java Course.java

Run the program using the following command:

	java Main

This will execute the Main class, which contains the main method that serves as the entry point for the program.

Note: Make sure that you have Java Development Kit (JDK) installed on your system, and that the java and javac commands are added to your system's PATH variable.
